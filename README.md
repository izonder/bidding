# Task

## Pre-requisites

* Node - v10.15.0+
* Yarn - v1.13.0+

## Development

**NOTE!** You should connect `.editorconfig` and `.eslintrc` to your IDE.

For test run type:

```bash
$ yarn test
```

For check linting only type (or you can activate automatic linting in your IDE, recommended):
```bash
$ yarn lint
```
